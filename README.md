### Step 3: Combined Projects
- `examples/16_full_example_shop_system.md` - مثال کامل: فروشگاه

## Run

- هر فایل `.md` شامل:
  - سوال
  - راهنمای مرحله‌ای
  - جواب کامل (کد دارت)
- کد قابل اجرا در پوشه `dart` با همان شماره فایل موجود است.
- مثال: `examples/14_encapsulation_only.md` ↔ `dart/14_encapsulation_only.dart`
