# Dart for C Programmers

راهنمای سریع و کاربردی **Dart** برای برنامه نویس هایی که **C** بلد هستند.

این مخزن برای مرور تفاوت ها و قابلیت های اضافه Dart نسبت به C ساخته شده و در هر درس، علاوه بر مثال پایه Dart، مثال مرتبط با **Flutter** هم ارائه می شود.

## چه چیزی در این آموزش پیدا می کنید؟

- توضیح مفاهیم Dart با فرض آشنایی قبلی با C
- تمرکز روی تفاوت های مهم (نه تکرار مبانی عمومی برنامه نویسی)
- مثال های واقعی برای Flutter در کنار هر مبحث
- نکات عملی برای استفاده در پروژه های واقعی

## پیش نیاز

- آشنایی با زبان C
- آشنایی اولیه با مفاهیم برنامه نویسی (متغیر، شرط، حلقه، تابع)

## مخاطب این آموزش

- برنامه نویس های C که می خواهند سریع وارد Dart شوند
- توسعه دهنده هایی که قصد شروع Flutter دارند
- کسانی که دنبال یک مرجع مرور سریع هستند

## ساختار آموزش

هر درس با یک الگوی ثابت نوشته می شود:

1. توضیح کوتاه مفهوم
2. تفاوت با C
3. مثال پایه در Dart
4. مثال کاربردی در Flutter
5. تمرین کوتاه (اختیاری)

## مسیر پیشنهادی مطالعه

- اول درس ها را به ترتیب بخوانید
- مثال Dart را اجرا کنید
- بعد همان مفهوم را در مثال Flutter بررسی کنید
- نکات کلیدی را برای مرور سریع علامت بزنید

## اجرای مثال ها

### اجرای Dart

```bash
dart run
```

### اجرای Flutter

```bash
flutter run
```

> بسته به ساختار مثال ها، ممکن است برای هر درس پوشه یا فایل جداگانه اضافه شود.
